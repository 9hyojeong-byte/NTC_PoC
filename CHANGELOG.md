# NTC PoC — 개선 내역 문서

> 초기 버전 대비 현재 버전에서 변경·개선된 사항을 영역별로 정리합니다.

---

## 1. 학습 활동 범위 최적화

| 활동 | 초기 버전 | 현재 버전 |
|------|-----------|-----------|
| 단어 퀴즈 | 기사 전체 단어 출제 | **4문항**으로 고정 |
| 문장 만들기 | 기사 전체 문장 출제 | **2·5·7·8번 문장** (4문장) 고정 |
| 녹음 | 기사 전체 문장 녹음 | **3·6번 문장** (2문장) 고정 |

학습 부담을 줄이고 핵심 문장에 집중할 수 있도록 각 활동의 출제 범위를 대폭 축소하였습니다.

---

## 2. 과제 발행 방식 변경

- **초기**: 기사마다 발행 날짜(at) 필드를 개별 관리
- **현재**: 날짜 개념 제거 → **주(週) 단위 발행** 방식으로 통일  
  - "현재 과제 / 직전 과제" 구분 폐지
  - 학습 현황 화면에서 반에 발행된 **모든 기사**를 한눈에 표시

---

## 3. 선생님 화면 — 학습 현황

### 3-1. 반별 기사 표시 방식 개선
- 초기: 현재/직전 과제 2개만 표시
- 현재: 반에 발행된 **전체 기사 목록** 표시 + 기사별 완료 학생 수 확인 가능

### 3-2. 난이도 표시 방식 통일
- 초기: "기초", "기본" 등 레이블 텍스트
- 현재: **렉사일(Lexile) 범위** 표기로 통일 (예: `100L–400L`, `610L–800L`)

### 3-3. 기사 수동 배정 기능
- 각 반 카드 우측 상단 **[+ 기사 배정]** 버튼 추가
- 모달에서 난이도·주제 필터 적용 후 기사 배정/취소 가능

### 3-4. 기사 회수 기능
- 기사 상세 모달(클릭) 내부 회색 기사 카드에 **회수 버튼** 추가
- 확인 후 해당 반 전체 학생의 배정 취소

### 3-5. 진행 현황 표기 개선
- "N명 미완료" → **"N명 진행중"** (회색 텍스트)

---

## 4. 선생님 화면 — 반 관리

### 4-1. 탭 분리
- 초기: 학습 현황 + 학생 관리가 한 화면에 스크롤로 공존
- 현재: **[학습 현황] / [반 관리]** 탭으로 완전 분리

### 4-2. 반 추가 기능 신설
- 반 이름, 기사 레벨(Kinder·Kids·Junior·Times), 수업 주기(주2·3·5회) 설정
- **LEVEL_FREQ_SEQS** 테이블 기반으로 신규 학생 입반 시 기사 자동 배정

### 4-3. 반 설정 편집 기능
- 각 반 카드 우측 **⚙ 설정 버튼** 추가
- 반 이름·난이도·수업 주기를 언제든 수정 가능

### 4-4. 반 카드 정보 표시 강화
- 반 이름 옆에 **난이도 배지 + 수업 주기** 나란히 표시
- BANDS 색상 기준으로 반별 컬러 코딩 (학습 현황과 동일 색상 적용)

### 4-5. 기본 반 구성 정리
- 초기: 입문반·기초반·기본반·심화반 4개 반 기본 제공
- 현재: **입문반(Kinder)·기본반(Junior)** 2개 반만 기본 제공 (기초반·심화반 제거)

---

## 5. 학생 등록 모달 개선

- **레벨 테스트로 추천받기** 탭 제거 → 직접 선택 방식만 유지
- 반 목록 필터링: 학생이 있거나 선생님이 직접 생성한 반만 노출
- 신규 학생 입반 시 해당 반의 기사를 **즉시 자동 배정**

---

## 6. 학생 상세 모달 신설

- 반 관리 화면에서 학생 이름 클릭 시 **개인별 학습 현황 테이블** 팝업
  - 배정된 모든 기사별 5단계 진행 상태 표시
  - 단어 퀴즈·문장 만들기 점수(N/M 형식) 표시
  - 제출된 **녹음 파일 직접 재생** 가능

---

## 7. 기사 배정 모달 UI 개선

- 난이도 표시: 컬러 배지 → **회색 렉사일 범위 텍스트**로 변경
- 주제 태그: 회색 배경 → **난이도 컬러 연동 컬러 태그**로 변경

---

## 8. 전체 UI / UX 정리

### 8-1. 헤더 슬림화
- 높이 56px → **44px** 축소, 로고 크기 비례 축소
- 상단 통계 카드(전체 배정·학습 완료·진행중·미시작) 제거
- 시드데이터 토글·초기화 버튼·선생님 얼굴 아이콘·이모지 제거
- 선생님/학생 전환 토글만 유지

### 8-2. 학생 화면 레이아웃 개선

| 항목 | 초기 | 현재 |
|------|------|------|
| 학생 선택 드롭다운 | 상단 헤더 내 | "나의 과제" 제목 우측 |
| 과제 카드 | 이미지·제목·버튼 한 줄 | 이미지+제목 상단 / 학습 버튼 하단 분리 |

### 8-3. 학습 단계 탭바 개편
- 기존 텍스트 나열 → **버튼 형태 스타일**로 변경
- 넓은 화면: 6개 단계 **한 줄** 배열
- 좁은 화면(≤ 520px): **3+3 두 줄** 자동 전환 (텍스트 줄내림 없음)
- 읽기 단계에서 탭바 위치: 히어로 이미지 아래 → **히어로 이미지 위**로 이동 (모든 단계 동일 위치)

### 8-4. 읽기 화면 컨트롤 반응형
- 넓은 화면: 번역보기·듣기·속도조절 한 줄
- 좁은 화면(≤ 480px): 번역보기+듣기 1행 / 속도조절 버튼 2행 분리

### 8-5. 기타 UX 수정
- 문장 만들기 단계에 **← 과제 목록** 버튼 추가
- 학습 완료 화면 녹음 제출 결과: 전체 문장 수 → **2문장**으로 수정

## 9. 데이터 구조 개선

- `CLASS_DEFAULT_AT` 제거, 날짜 필드(`at`) 전면 삭제
- `LEVEL_FREQ_SEQS` 추가: 레벨 × 수업 주기 → 자동 배정 기사 목록 정의
- `cls.level` 필드 추가: 동적 생성 반의 레벨 정보 저장
- localStorage 마이그레이션: 기존 기초반(c2)·심화반(c4) 데이터 자동 제거
- 입문반(c1)·기본반(c3) 기본 레벨 자동 마이그레이션
